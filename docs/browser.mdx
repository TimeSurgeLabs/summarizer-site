# Browser Extension

Follow these simple steps to quickly set up Summarize and begin enjoying concise and accurate summaries of your favorite YouTube videos.

## Step 2: Create an Account

Create an account at [https://app.summarize.cc/register](https://app.summarize.cc/register) to unlock the full functionality of Summarize and enjoy a seamless, personalized experience.

## Step 3: Log in to the Extension

Click on the Summarize extension icon in your browser toolbar and enter your login credentials to log in.

## Step 4: Get an OpenAI API Key

To enable the AI-based summarization feature, you'll need an OpenAI API key. Follow the instructions in this guide to obtain one: [How to get an OpenAI API key](https://www.onmsft.com/how-to/how-to-get-an-openai-api-key/)

## Step 5: Paste the API Key into the Extension

Once you have your OpenAI API key, click on the Summarize extension icon, paste the key into the appropriate field, and hit "Save" to authenticate with the OpenAI API. Your API key will be securely stored and never leave your browser.

## Step 6: Start Browsing YouTube and Enjoy Summaries!

Now you're all set! Start browsing YouTube and watch as Summarize generates concise summaries for the videos you're interested in. Save time and get straight to the heart of the content you love.

Happy summarizing!
